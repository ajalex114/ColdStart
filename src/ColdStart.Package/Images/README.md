# ColdStart — Placeholder Store Assets
#
# The MSIX package requires PNG images at specific sizes for the Microsoft Store.
# Replace these placeholder files with actual branded images before Store submission.
#
# Required images:
#   StoreLogo.png         — 50x50    (Store listing tile)
#   Square44x44Logo.png   — 44x44    (App list icon)
#   Square71x71Logo.png   — 71x71    (Small tile)
#   Square150x150Logo.png — 150x150  (Medium tile)
#   Wide310x150Logo.png   — 310x150  (Wide tile)
#   Square310x310Logo.png — 310x310  (Large tile)
#   SplashScreen.png      — 620x300  (Splash screen)
#
# Tools to generate these:
#   - https://www.pwabuilder.com/imageGenerator (auto-generates all sizes from one image)
#   - Any image editor — export at the sizes above
#
